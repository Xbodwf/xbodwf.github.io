--- title: 使用Vite+Typescript+React构建Github Pages description: Use framework to build gh-pages date: 2025-8-7 updated: 2025-8-7 category: Develop/Server tags: [React,Typescript,Frontend,github Pages] authors: - name: Xbodwf email: xbodwf@outlook.com github: Xbodwf --- # Vite+Typescript+React ## 开始 首先我们先选择一个位置,打开终端，输入 ``` pnpm create vite@latest ``` 稍后pnpm会提示你输如入项目名，输入： ```bash {你的Github用户名}.github.io ``` 选择语言时选择Typescript 框架选择react 之后你会得到一个xxx.github.io的文件夹， 里面存放了一些基础的文件，整个template 接下来我们需要前往Github创建一个仓库 命名同刚才的文件夹。 创建完仓库后，回到之前的终端，依次执行以下步骤: ```bash git init git commit -m "first commit" git branch -M main git remote add origin git仓库地址 git push -u origin main ``` 该操作是将你的本地代码提交到git仓库上。 push完成后，在`package.json`中添加如下内容: ```json { ...其他内容, "scripts": { ...其他脚本, "build": "tsc -b && vite build" "deploy": "gh-pages -d dist", "predeploy": "pnpm build" } } ``` 终端输入以下内容并运行: ```bash pnpm install gh-pages --save-dev ``` 然后你就可以开始开发了。 1.需要更新仓库代码时像平时一样push 2.需要更新Github Pages时`pnpm run deploy`即可。 3*.别忘了在仓库设置中修改github Pages的分支为gh-pages分支(踩过坑)
