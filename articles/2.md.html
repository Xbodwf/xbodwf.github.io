""" if n == 0: return 1 else: return n * factorial(n-1) # 测试函数 print(f"5! is {factorial(5)}") ``` ``` 这是一个没有指定语言的普通文本块。 它不会有任何语法高亮。 ``` ## 6. 水平分割线 (Horizontal Rule) 可以通过三个或更多的星号、减号或下划线来创建水平分割线。 --- *** ___ ## 7. 链接 (Links) 这是一个行内链接：[访问 Google](https://www.google.com "Google 搜索引擎"). 这是一个引用式链接：[维基百科][1]. 你可以在文档的其他地方定义这个链接。 URL 和邮箱地址可以被自动转换成链接： [1]: https://www.wikipedia.org/ "维基百科官网" ## 8. 图片 (Images) ![Markdown Logo](https://markdown.com.cn/assets/img/logo.png "Markdown 标志") *图片来自 markdown.com.cn* 图片也可以作为链接： [![一只可爱的小狗](https://picsum.photos/id/237/300/100)](https://picsum.photos/) *(点击上方图片跳转)* ## 9. 表格 (Tables) | 表头 1 (左对齐) | 表头 2 (居中) | 表头 3 (右对齐) | | :-------------- | :-----------: | --------------: | | 内容 1 | 内容 2 | 内容 3 | | 长内容单元格 | **粗体** | *斜体* | | `代码` | 短内容 | | ## 10. 扩展语法 (Extended Syntax) 以下语法可能不是所有解析器都支持，但在 GFM (GitHub Flavored Markdown) 中很常见。 ### 脚注 (Footnotes) 这是一个带有脚注的句子。[^1] 另一个脚注在这里。[^longnote] ### 定义列表 (Definition Lists) Markdown : 一种轻量级的标记语言，用于使用纯文本格式的编辑器创建格式化文本。 HTML : 超文本标记语言，用于创建网页的标准标记语言。 ### Emoji (表情符号) :smile: :rocket: :tada: :computer: ## 11. 转义字符 (Escaping Characters) 如果你想显示 Markdown 的特殊字符本身，而不是让它们触语法，可以使用反斜杠 `\` 进行转义。 \*这不是斜体\* \# 这不是标题 \` 这不是代码 \` \[ 这不是链接 \] --- [^1]: 这是第一个脚注的解释。 [^longnote]: 这是一个更长的脚注，可以包含多个段落和代码。 代码也可以在脚注中：`code_in_footnote()` 甚至是列表： - 列表项1 - 列表项2
